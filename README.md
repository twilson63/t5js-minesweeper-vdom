# minesweeper virtual-dom

This is a simple implementation of minesweeper for the virtual-dom using:

- hyperscript-helpers
- snabbdom
- mostjs
- ramdajs

## Development Setup

```
npm install
npm start
```

open browser to http://localhost:9966
